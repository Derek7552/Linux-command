# Linux常用命令学习大纲

## 学习目标
掌握Linux系统中最常用的命令，提高命令行操作效率和系统管理能力。

## 学习进度建议
- 每天学习1-2个模块
- 每个命令都要动手练习
- 理解命令参数和选项的含义
- 掌握管道和重定向的使用

---

## 模块1：文件和目录操作命令

### 1.1 目录浏览和切换
**命令：`ls` (list)**
- 用途：列出目录内容
- 常用选项：
  - `ls -l`：详细列表格式
  - `ls -a`：显示隐藏文件
  - `ls -h`：人性化显示文件大小
- 示例：`ls -lah`

**命令：`cd` (change directory)**
- 用途：切换工作目录
- 特殊用法：
  - `cd ..`：上级目录
  - `cd ~`：用户主目录
  - `cd -`：上一个目录

**命令：`pwd` (print working directory)**
- 用途：显示当前工作目录

### 1.2 文件和目录操作
**命令：`mkdir` (make directory)**
- 用途：创建目录
- 常用选项：`mkdir -p` (递归创建)

**命令：`rmdir` (remove directory)**
- 用途：删除空目录

**命令：`touch`**
- 用途：创建空文件或更新文件时间戳

**命令：`cp` (copy)**
- 用途：复制文件和目录
- 常用选项：
  - `cp -r`：递归复制目录
  - `cp -i`：交互式复制（覆盖前询问）

**命令：`mv` (move)**
- 用途：移动或重命名文件

**命令：`rm` (remove)**
- 用途：删除文件和目录
- 常用选项：
  - `rm -r`：递归删除
  - `rm -i`：交互式删除
  - `rm -f`：强制删除

### 1.3 文件查看和编辑
**命令：`cat` (concatenate)**
- 用途：查看文件内容

**命令：`less` 和 `more`**
- 用途：分页查看文件内容
- `less` 支持上下滚动，`q`退出

**命令：`head` 和 `tail`**
- 用途：查看文件开头或结尾
- 常用选项：`-n 10` 显示10行

### 练习建议
1. 创建目录结构：`mkdir -p project/docs/images`
2. 复制和移动文件练习
3. 使用`ls`的不同选项查看文件信息
4. 练习文件删除（小心使用`rm -rf`）

## 模块2：文本处理命令

### 2.1 文本搜索
**命令：`grep` (global regular expression print)**
- 用途：在文件中搜索文本
- 常用选项：
  - `grep -i`：忽略大小写
  - `grep -n`：显示行号
  - `grep -r`：递归搜索目录
- 示例：`grep -rn "error" /var/log/`

**命令：`find`**
- 用途：在目录树中查找文件
- 常用用法：
  - `find . -name "*.txt"`：按名称查找
  - `find . -type f -size +1M`：查找大于1MB的文件

### 2.2 文本编辑和处理
**命令：`sed` (stream editor)**
- 用途：流编辑器，用于文本替换
- 示例：`sed 's/old/new/g' file.txt`

**命令：`awk`**
- 用途：文本处理和报告生成器
- 示例：`awk '{print $1}' file.txt`

**命令：`sort`**
- 用途：对文本行排序

**命令：`uniq`**
- 用途：去除重复行

**命令：`wc` (word count)**
- 用途：统计文件行数、字数、字符数

### 练习建议
1. 使用`grep`搜索系统日志
2. 练习文本替换操作
3. 统计文件中的单词频率
4. 查找特定类型的文件

## 模块3：系统管理命令

### 3.1 进程管理
**命令：`ps` (process status)**
- 用途：显示进程信息
- 常用选项：`ps aux`

**命令：`top` 和 `htop`**
- 用途：实时显示系统进程

**命令：`kill`**
- 用途：终止进程
- 示例：`kill -9 PID`

**命令：`bg`、`fg` 和 `jobs`**
- 用途：作业控制（后台、前台运行）

### 3.2 系统资源监控
**命令：`df` (disk free)**
- 用途：显示磁盘使用情况

**命令：`du` (disk usage)**
- 用途：显示目录磁盘使用情况
- 常用选项：`du -sh *`

**命令：`free`**
- 用途：显示内存使用情况

**命令：`uptime`**
- 用途：显示系统运行时间

**命令：`who` 和 `w`**
- 用途：显示登录用户

### 练习建议
1. 监控系统资源使用情况
2. 练习进程的启动和终止
3. 查看系统日志文件
4. 了解系统负载情况

## 模块4：权限管理命令

### 4.1 文件权限
**命令：`chmod` (change mode)**
- 用途：修改文件权限
- 两种方式：
  - 数字方式：`chmod 755 file`
  - 符号方式：`chmod u+x file`

**命令：`chown` (change owner)**
- 用途：修改文件所有者

**命令：`chgrp` (change group)**
- 用途：修改文件所属组

### 4.2 用户和组管理
**命令：`sudo`**
- 用途：以超级用户权限执行命令

**命令：`su`**
- 用途：切换用户

**命令：`passwd`**
- 用途：修改密码

### 练习建议
1. 修改文件权限并观察效果
2. 练习使用`sudo`执行系统命令
3. 了解文件权限的数字表示法

## 模块5：网络相关命令

### 5.1 网络连接测试
**命令：`ping`**
- 用途：测试网络连通性

**命令：`traceroute`**
- 用途：显示数据包的路由路径

**命令：`nslookup` 或 `dig`**
- 用途：DNS查询

### 5.2 文件传输
**命令：`wget`**
- 用途：从网络下载文件

**命令：`curl`**
- 用途：传输数据的工具
- 示例：`curl -O http://example.com/file.zip`

**命令：`scp` (secure copy)**
- 用途：安全复制文件（SSH）

**命令：`rsync`**
- 用途：高效的文件同步工具

### 5.3 远程连接
**命令：`ssh` (secure shell)**
- 用途：安全远程登录
- 示例：`ssh user@hostname`

### 练习建议
1. 测试网络连接
2. 下载文件练习
3. 远程服务器连接（如果有条件）

## 模块6：包管理和软件安装

### 6.1 常见包管理器
**Debian/Ubuntu：`apt`**
- `apt update`：更新包列表
- `apt install package`：安装软件
- `apt remove package`：卸载软件

**RedHat/CentOS：`yum` 或 `dnf`**
- `yum install package`

**Arch Linux：`pacman`**
- `pacman -S package`

### 练习建议
1. 更新系统包列表
2. 安装常用软件
3. 卸载不需要的软件

## 模块7：其他实用命令

### 7.1 命令历史和别名
**命令：`history`**
- 用途：显示命令历史

**命令：`alias`**
- 用途：创建命令别名
- 示例：`alias ll='ls -l'`

### 7.2 压缩和解压
**命令：`tar`**
- 用途：打包和压缩文件
- 常用选项：
  - `tar -czf archive.tar.gz file`：压缩
  - `tar -xzf archive.tar.gz`：解压

**命令：`gzip`、`bzip2`、`xz`**
- 用途：文件压缩工具

### 7.3 系统信息
**命令：`uname`**
- 用途：显示系统信息

**命令：`date`**
- 用途：显示或设置日期时间

**命令：`cal`**
- 用途：显示日历

### 练习建议
1. 创建有用的命令别名
2. 练习文件压缩和解压
3. 查看系统信息
