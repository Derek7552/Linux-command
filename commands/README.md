## 🚀 命令速查表

### 文件操作 (高频)
```bash
ls -la          # 查看详细文件列表
cd /path        # 切换目录
cp file1 file2  # 复制文件
mv file1 file2  # 移动/重命名
rm -rf dir/     # 删除目录
```

### 系统监控 (必备)
```bash
ps aux          # 查看进程
top             # 系统监控
df -h           # 磁盘使用
free -h         # 内存使用
```

### 网络操作 (常用)
```bash
ping host       # 测试连通性
ssh user@host   # 远程登录
scp file host:  # 文件传输
wget url        # 下载文件
```
