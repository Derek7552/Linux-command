# 📚 学习指南 (Guides)

本目录包含Linux命令学习的各种指南和教程文档。

## 📁 目录结构

```
guides/
├── stages/           # 📖 分阶段学习教程
│   ├── 阶段1-基础入门.md    # 文件和目录操作基础
│   ├── 阶段2-文本处理.md    # 文本处理和编辑
│   ├── 阶段3-系统管理.md    # 系统监控和管理
│   └── 阶段4-网络与高级应用.md # 网络和高级技巧
├── quick-start/      # 🚀 快速入门指南
└── roadmap/          # 🗺️ 学习路线图
```

## 📚 学习路径推荐

### 🏃‍♂️ 新手学习路径
1. **阶段1**: 掌握基础的文件和目录操作
2. **阶段2**: 学习文本处理和数据操作
3. **阶段3**: 了解系统管理和监控
4. **阶段4**: 探索网络和高级应用

### 🔄 查漏补缺路径
- 需要特定技能时，直接查看相应阶段
- 每个阶段都是相对独立的模块

## 💡 使用建议

- **循序渐进**: 建议按照阶段顺序学习
- **实践为主**: 每个阶段都包含动手练习
- **查阅手册**: 遇到不清楚的命令可使用 `man` 命令
- **记笔记**: 在学习过程中记录心得体会

## 🎯 各阶段目标

| 阶段 | 主要内容 | 学习目标 | 预期时间 |
|------|----------|----------|----------|
| 阶段1 | 文件操作、特殊符号 | 建立命令行基础 | 2-3天 |
| 阶段2 | 文本处理、搜索编辑 | 掌握数据处理技能 | 3-4天 |
| 阶段3 | 系统监控、管理权限 | 学习系统维护 | 4-5天 |
| 阶段4 | 网络应用、高级技巧 | 掌握综合应用 | 5-7天 |

## 📖 学习资源

- **在线平台**: [Linux命令行练习](https://cmdchallenge.com/)
